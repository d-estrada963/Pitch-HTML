Motivations:

  • Enriching the experience of music listeners
  • Creating a single stop for finding opinions about songs and discovering new music based on personal preferences and musical taste
  • Creating a web application that is both simple but serves a practical function for a wide audience. Lots of people listen to music.
  • Developing a better understanding of SQL, HTML, and CSS which will be required to fine tune and perfect the project.

Deliverables:

  • Basic UI Layout
    1. Home Page
      • Navigation bar: HOME / GENRES / TOP HITS / ARTISTS / SEARCH / ABOUT US / REGISTER
      • Search bar (search music database and return results)
      • TOP HITS (can be based on the rating system that we use)
    2. Song Page
      • Ability to share on social media
      • Links to music
      • Ratings, reviews
      • Misc. info ("fun facts)
  • Rating System
    • Will reuire basic math operations and star graphics
    • Continuously interacts with the database to ensure each rating/review is recorded
  • Music Database
    • Web Scraper software for data mining or API or Manually if we are creating a short database.
  • User Account Functionality
    • Ability for users to register and interact on the website
    • Username/password combination (another database required probably)
